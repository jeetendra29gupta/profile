My Work and Project


Some Official Project

EGL.

Description: EGL is a Network Management System for monitoring One97 servers and services deployed in Networks. EGL software provides UI views for an effective, efficient way to manage servers without the need to spend time checking each individual server or service.

Language and Tools: JAVA, Eclipse, OpenNMS, Struts2, Hibernate, Log4J, Postgres, SNMP4J.

Hardware Inventory panel.

Description: Dynamic user interface developed in JAVA which fetches data/record from database which was pushed by a python program to display hardware inventory details like RAM, CPU, Processor, hard-disk of all nodes in monitoring.

Language and Tools: JAVA, Python, Eclipse, Hibernate, SNMP4J, Log4J, MYSQL.

Extended-agent.

Description: The application was designed and developed for monitoring Avaya's and international operator's servers. It was written in python and deployed on every server. The main task was to collect hardware and system details (RAM, CPU, Processor, hard-disk) and send an alert when the threshold limit is breached.

Language and Tools: Python, Pycharm, Psutil, logging, SNMP trap.

Push-agent and Agent-less.

Description: This project was developed for monitoring Payment-bank servers and Ticket-new servers using Java. This application collects the server's data and converts into XML then push into OpenNMS tool.

Language and Tools: Java, Eclipse, Log4J, Maven, jaxb-api(XML), oshi-core(JNA), OpenNMS, JSCH.
